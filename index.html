🧠 어떻게 만들었나

핵심 기능: 인디 개발자가 2D 아바타를 통해 기획, 아트, 코드 등 각 AI 부서와 협업하며 게임을 빌드하는 '오피스 컨셉' 제작 도구입니다.
기술 방식: Phaser.js로 구축된 웹 환경에서 사용자가 배치한 오브젝트를 LLM이 분석하여 게임 스크립트를 자동 생성하고, 이미지 생성 AI가 스프라이트를 즉석 구현합니다.
사용 방법: 사용자가 2D 오피스로 '출근'하여 각 부서 AI에게 명령을 내리면, WYSIWYG 편집기를 통해 코딩 없이 실시간으로 게임이 완성됩니다.
